# 🤖 EduBot – AI Student Learning Assistant

EduBot is a Python-based desktop learning assistant that helps students:
- 📚 Explain study topics from PDFs  
- 🧠 Generate quizzes  
- 🌦 Get live weather updates  
- 📰 Fetch latest news  
- 💬 Show motivational quotes  
- 🎵 Play YouTube music playlists  
- ⏰ Set reminders & alarms  

---

## 🧩 Features
- GUI-based (Tkinter)
- No external AI key required
- Works fully offline except for APIs

---
